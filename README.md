# ExecuMind AI 🚀

## 📌 Overview
ExecuMind AI is an intelligent task execution and decision-support system designed to transform unstructured thoughts into actionable plans, helping users improve focus, productivity, and decision-making.

---

## 🎯 Problem Statement
Students and professionals often struggle to:
- Organize scattered thoughts
- Prioritize tasks effectively
- Maintain consistency and execution

---

## 💡 Solution
ExecuMind AI provides:
- Structured task extraction from free-form input
- AI-based prioritization and planning
- Time-blocked scheduling
- Execution tracking and feedback

---

## 🔍 Features
- 🧠 NLP Engine: Converts raw thoughts into structured tasks  
- 📊 Decision Engine: Prioritizes tasks intelligently  
- 📅 Planner: Generates time-blocked schedules  
- 🤖 AI Mentor: Provides real-time guidance  
- 🎯 Tracker: Tracks progress, XP, and streaks  

---

## 🛠 Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- AI Integration: Google Gemini API

---

## 🧠 System Architecture
Frontend (HTML/JS)
↓
Flask Backend (API Routes)
↓
AI Processing Layer (Gemini / Fallback)
↓
Response → UI Update

---

## ⚙️ How It Works
1. User inputs tasks or thoughts  
2. NLP engine structures them  
3. Decision engine prioritizes tasks  
4. Planner generates schedule  
5. AI mentor guides execution
